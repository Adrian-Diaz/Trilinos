00029 /*
00030  * symbolicFactorization_impl() performs no action other than making sure that
00031  * Factorization is performed.  numericFactorization_impl() performs the
00032  * factorization but no solve (right hand side is set to 0 vectors) reuses
00033  * factors only if ReuseFactorization_ is set.
00034  *  
00035  *  If FactorizationOK_() && ReuseSymbolic_ 
00036  *     reFactor()
00037  *  else
00038  *     factor() 
00039  *
00040  * solve() 
00041  *
00042  * factor() does everything from scratch:
00043  *  - Redistributes the data if necessary
00044  *  - Deletes any data structures left over from the previous call to
00045  *    Factor()
00046  *  - Copies the data into the format that SuperLU wants it
00047  *  - Calls dgssvx to factor the matrix with factor set to true
00048  * reFactor()
00049  *  - Redistributes the data if necessary
00050  *    + Attempting to check to make sure that the non-zero structure is
00051  *      unchanged
00052  *  - Copies the data into the format that SuperLU already has it
00053  *    FIRST PASS - assert( false ) 
00054  */
