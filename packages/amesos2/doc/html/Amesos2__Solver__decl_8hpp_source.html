00057 /* This is the base class to be used in a *statically* polymorphic

00060  * In Amesos2_Superlu.hpp:

00063  * Each concrete solver will implement several private subfunction
00064  * that will be called within the common code for each function.

00066  * In addition, the Amesos2::Solver static base class provides a convenient
00067  * way to handle the template parameters and the private typedef-ing (with
