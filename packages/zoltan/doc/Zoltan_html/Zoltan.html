Project Goals:
  •  Goal:  Develop a general-purpose tool providing dynamic load-balancing to a variety of applications.
    • Zoltan's function-call interface supports calls to the load-balancing library from the application, allowing the partitioning algorithms to be used dynamically or statically.
    • An object-oriented interface separates Zoltan's data structures from those of the application, allowing many different applications to use the library.
  • Goal:  Enable research into dynamic load-balancing algorithms.
    • The library is a tool-kit containing several different load-balancing algorithms.
    • Applications can experiment with different algorithms, enabling comparisons of partition quality and load-balancing costs.
    • Zoltan's interface allows easy development of new algorithms.
  • Goal: Provide support for data migration.
    • Zoltan's migration-help tools can handle all communication related to data migration, greatly simplifying migration for the application developer.
    • Applications provide only a few routines to pack and unpack data from their data structures.
